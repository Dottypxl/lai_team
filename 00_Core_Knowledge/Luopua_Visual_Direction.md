# Luopua Visual Direction

## 1. Purpose

This file collects visual direction for Luopua by Kirsi.

It is a growing design judgement file.

It does not replace `LUOPUA-BRAND.md`.

The master brand file defines the foundation.

This file helps make visual choices inside that foundation.

## 2. Core visual idea

Luopua means letting go.

The visual direction should support that.

Let go of:

* visual noise
* generic templates
* decorative depth
* over-polished perfection
* too many styles at once
* design choices that do not carry the story

Keep:

* calm
* clarity
* nature
* space
* texture
* hand-drawn marks
* useful imperfection
* strong typography
* soft structure

## 3. Master brand foundation

For Luopua by Kirsi, the shared foundation is:

* hand-drawn master mark
* misty off-white background `#F4F6F1`
* deep forest text `#2E3B2C`
* sage `#8DA383`
* light sage `#D8E0D0`
* muted text `#6B7A66`
* headings and wordmarks: Vollkorn 500–600
* body text: Inter 400/700
* soft rounded shapes
* hand-drawn line illustrations
* no emojis in UI or printed materials

Do not casually replace these.

They are the house.

## 4. Product room flexibility

Products may have their own:

* short Finnish nature-word name
* one accent colour
* tagline
* product-specific illustration set
* layout emphasis
* mood within the master brand

Products should not have:

* completely separate logo
* unrelated typography system
* unrelated colour palette
* visual language that breaks the Luopua house

A product can have its own room.

It should still belong to the same house.

## 5. Typography direction

### Master brand rule

For Luopua by Kirsi, use the defined foundation:

```text
Headings and wordmarks: Vollkorn 500–600
Body: Inter 400/700
```

This keeps the brand recognizable across services, products, documents, and future tools.

### Exploration rule

For client work, campaign experiments, future products, or visual studies, do not default to generic fonts as the first choice.

Use generic fonts only when clarity, speed, accessibility, or technical constraints matter more than distinctiveness.

Typography should signal:

* quality
* character
* intention
* the story behind the work

Do not automatically default to:

* Inter
* Roboto
* Open Sans
* Lato
* system fonts

These can be useful.

They should not be automatic.

## 6. Preferred typography directions for exploration

### Code / structured

Use when the work needs structure, systems, notes, tools, or technical clarity.

Examples:

* JetBrains Mono
* Fira Code
* Space Grotesk

### Editorial

Use when the work needs depth, reading rhythm, reflection, or essay-like presence.

Examples:

* Playfair Display
* Crimson Pro
* Newsreader

### Technical / clear

Use when the work needs clarity, accessibility, and calm function.

Examples:

* IBM Plex family
* Source Sans 3

### Distinctive

Use when the work needs character but should still stay readable.

Examples:

* Bricolage Grotesque

## 7. Typography pairing principle

Use contrast with purpose.

Strong pairings can include:

* serif + geometric sans
* display font + monospace
* one variable font used across clear weight differences

One distinctive font is enough when used well.

Do not make typography compete with the message.

## 8. Type scale principle

Use scale decisively.

Prefer:

* clear size jumps over minor differences
* clear weight contrast over tiny variation
* strong hierarchy over decorative styling

Possible contrast:

```text
light weight 100/200
against
heavy weight 800/900
```

Use this only when it fits the context.

Luopua itself should stay calm, not shout through type.

## 9. Before choosing fonts

Ask:

* Is this calm or expressive?
* Rural or urban?
* Editorial or functional?
* Premium or accessible?
* Warm or sharp?
* Natural or technical?
* Service or product?
* Master brand or product room?
* Does this font help the story?
* Does this font create unnecessary noise?

Fonts should support the story.

Not decorate it.

## 10. Layout direction

Luopua layouts should feel:

* calm
* spacious
* soft
* readable
* intentional
* grounded
* not empty for the sake of being minimal

Use:

* generous spacing
* clear hierarchy
* soft rounded corners
* calm cards
* fewer sections
* one clear action per view
* enough white or misty space for the thought to breathe

Avoid:

* crowded layouts
* too many boxes
* too many calls to action
* over-layered textures
* template-like hero sections
* visual tricks that do not help understanding

## 11. Image direction

Images should feel:

* documentary
* natural
* tactile
* close to real work
* slightly imperfect when useful
* connected to place, process, or material

Good subjects:

* handwritten notes
* crossed-out words
* paper
* wood
* forest
* paths
* small workspaces
* hands at work
* rural details
* old rooms
* natural light
* objects with use and texture

Avoid:

* stock-photo business scenes
* perfect laptop-and-coffee clichés
* fake wellness imagery
* generic Nordic lifestyle clichés
* over-staged “calm”
* images that look expensive but say nothing

## 12. Illustration direction

Illustrations should follow the hand-drawn mark.

Use:

* single colour
* line-based SVGs
* round line caps
* organic curves
* slight irregularity
* simple nature references
* useful symbolic clarity

Avoid:

* emojis
* filled icon packs
* perfect geometry
* decorative symbols from another culture
* complex illustrations that become the point

## 13. Texture direction

Texture is allowed when it supports the work.

Use texture for:

* paper
* old wood
* hand marks
* natural surface
* visible process
* printed material
* lived place

Do not use texture to fake depth.

Texture should feel like the work has been touched by a person.

## 14. Philosophy as visual practice

Japanese philosophies can inform visual choices only as practice.

Do not make them the visual theme.

### Wabi-sabi

Keep useful imperfection.

Visual behaviour:

* hand-drawn mark
* uneven line
* lived texture
* old surface
* human trace

### Ma

Leave space.

Visual behaviour:

* margins
* silence
* breathing room
* fewer elements
* one thought per section

### Kanso

Remove excess.

Visual behaviour:

* cut decoration
* simplify hierarchy
* reduce choices
* one clear action

### Ikigai

Check fit.

Visual behaviour:

* does this design fit the person, work, need, and livelihood?

## 15. Visual decision rule

Before adding a visual element, ask:

* What does this help the reader understand?
* Does this belong to Luopua by Kirsi?
* Is this master brand, product room, or experiment?
* Is it carrying meaning or adding decoration?
* Can it be simpler?
* Can it have more space?
* Can it feel more human?
* Can one visual idea be removed?

## 16. Final rule

Visual direction is not styling.

It is choosing what the work needs to carry.
