## What changed in this rebrand

The brand centre is now simpler and more genuine:

```text
Luopua by Kirsi
= letting go of excess
= calm clarity in work and nature
= one house, many rooms
```

The Japanese philosophies can remain as inner working principles, but they should not become decorative public language.

Use them as behaviour:

- keep texture
- leave space
- remove excess
- test fit between person, work, need, and livelihood

Do not use them as surface styling or borrowed depth.

## What each file does

### Luopua_Brand_DNA.md

Defines the master brand, architecture, product logic, visual foundation, and philosophy boundary.

### Luopua_Voice_Guide.md

Defines how Luopua by Kirsi sounds across all helpers.

### Luopua_Audience_Guide.md

Defines the people Luopua serves and how to speak to them with respect.

### Luopua_Method.md

Defines the working method: let go, find the red thread, remove noise, build from own ground.

### Luopua_Offers.md

Defines services, products, and offer boundaries.

### Luopua_Case_Library.md

Defines source cases: Traktorijatzit, Suovanlahti, Saarnikumpu, Polku, portfolio work, and Luopua AI Team.

### Luopua_Forbidden_Language.md

Defines words, phrases, tactics, and tones to avoid.
