# Luopua Offers and Product Rooms

## 1. Purpose

This file defines current and emerging Luopua by Kirsi services and products.

Services use the master brand directly.

Products receive short Finnish nature-word names inside the same master brand house.

## 2. Offer logic

Luopua service offers should follow this order:

```text
Clarity
↓
Copy or content
↓
Website or AI system
↓
Care and refinement
```

Do not sell tools before clarity.

Do not sell ongoing support without boundaries.

## 3. Master brand services

Use:

```text
Luopua by Kirsi
```

for services such as:

- clarity sessions
- brand clarity
- website clarity
- web design
- content marketing
- coaching
- AI helper setup
- service and offer clarity

Do not create a product name for every service.

## 4. Product rooms

Products can use short Finnish nature words.

Current product:

```text
polku
```

Polku is a nature planner and journal.

Possible future product names:

- havu
- sammal
- lähde
- oksa
- pesä
- virta

Use a product name only when the thing is a repeatable product, tool, template, journal, app, planner, or clearly packaged asset.

Do not use product names for ordinary client services.

## 5. Entry services

### Luopua Clarity Session

Purpose:

A focused session for small businesses that need a clearer base before building more content, websites, offers, or AI tools.

Good for:

- scattered message
- unclear website
- content pressure
- AI outputs that sound generic
- new offer idea
- rural business visibility

Possible deliverables:

- red thread sentence
- audience notes
- offer summary
- message angles
- next-step recommendation
- short source text

Boundary:

Not a full brand strategy.

Not a website build.

Not social media management.

Not unlimited AI support.

### Website Clarity Review

Purpose:

A practical review of what a website is trying to carry.

Good for:

- unclear homepage
- outdated page
- offer confusion
- reader path issues
- before rebuilding a site

Possible deliverables:

- homepage clarity notes
- offer and audience observations
- reader path notes
- priority fixes
- next-step recommendation

Boundary:

Not a technical audit unless agreed.

Not full copy rewrite unless scoped separately.

## 6. Core services

### Brand Clarity Project

Purpose:

Create a grounded brand base for a small business, cultural project, or community-rooted organisation.

Good for:

- unclear brand voice
- scattered services
- new business direction
- website planning
- AI helper source material
- content direction

Possible deliverables:

- brand red thread
- audience clarity
- core message
- voice direction
- offer direction
- visual direction notes
- content angles
- next-step map

Boundary:

Not full visual identity unless included.

Not website build unless added.

Not ongoing content management.

### AI Helper Setup

Purpose:

Build one clear AI helper role for a specific business task.

Good for:

- content support
- client replies
- offer copy
- internal decision support
- project communication
- reducing repeated work

Possible deliverables:

- role instructions
- voice boundaries
- workflow file
- source file list
- example prompt
- use notes

Boundary:

One helper only unless larger scope is agreed.

Not full automation.

Not open-ended support.

Limited page structure.

## 8. Product: Polku

### Name

polku

### Meaning

Polku means path.

### What it is

Nature planner and journal.

### Tagline

Luonto kulkee mukanasi · A nature planner & journal

### Brand role

Polku is a product room inside Luopua by Kirsi.

It uses the master mark, shared typography, shared base palette, and its own puolukka accent.

### Accent

Puolukka:

```text
#9C4A41
```

Use for actions only.

### Feel

Polku should feel:

- natural
- organic
- calm
- soft
- rounded
- useful
- unhurried

### Boundary

Polku is not a productivity app wearing nature colours.

It is a calm planning and reflection product where nature travels with the user.

## 9. Possible care offers

### Monthly Clarity Care

Purpose:

Light support after a clarity or website project.

Could include:

- one review
- content prompts
- AI helper refinement
- small copy updates
- one monthly call

Boundary:

Not full social media management.

Not constant access.

### AI Helper Refinement

Purpose:

Improve an existing AI helper after testing.

Could include:

- output review
- instruction update
- new examples
- boundary edits
- source file update

Boundary:

One helper or agreed scope.

## 10. What Luopua does not sell as default

Avoid making these main offers:

- daily social media management
- cheap content production
- open-ended consulting
- full agency retainers without structure
- generic AI prompt packs
- logo-only design work
- websites without clarity work
- free strategy hidden inside sales calls
- products created only because the name sounds nice

## 11. Offer readiness check


## 12. Product readiness check

A product is ready when these are clear:

```text
Name:
Meaning:
User:
Use case:
Format:
Core promise:
Accent:
What it shares with Luopua:
What it owns:
Boundary:
```

Do not build a product only because the name is beautiful.

## 13. Offer language

Use:

- clarity base
- red thread
- website as tool
- own ground
- useful next step
- bring the messy version
- make the work easier to understand
- source material for AI
- calmer visibility
- let go of what hides the point

Avoid:

- high-converting
- irresistible
- dream clients
- elevate
- scale
- unlock
- transform
- effortless
- content machine

## 14. Final rule

A Luopua offer should reduce confusion before it asks for commitment.

A Luopua product should feel like one room in the same house.
