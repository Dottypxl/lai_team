# Kutsu – CTA Library

## 1. Purpose

This file gives Kutsu clear CTA options for Luopua.

A Luopua CTA should feel like a useful next step.

Not pressure.

Not urgency theatre.

Not a trick.

## 2. CTA rule

The CTA should match the reader's readiness.

Do not ask for a large commitment when the reader only needs a small next step.

## 3. CTA levels

### Low commitment

Use when the reader is curious but not ready to book.

Examples:

- Send me what feels unclear.
- Ask if this fits your situation.
- Tell me what your website is trying to carry.
- Bring the messy version.
- Start with one sentence.
- Send the rough draft.
- Ask about the next step.

### Medium commitment

Use when the reader understands the offer.

Examples:

- Book a clarity session.
- Request a website clarity review.
- Start with the red thread.
- Plan your AI helper.
- Send your website for review.
- Book the first project call.
- Request a Luopua review.

### High commitment

Use when the reader is ready for a larger project.

Examples:

- Request a proposal.
- Start the brand clarity project.
- Book the project start call.
- Ask about working together.
- Reserve a project slot.

Use high-commitment CTAs only when the page has done enough explaining.

## 4. CTAs by offer

### Luopua Clarity Session

Primary:

- Book a clarity session
- Start with the red thread
- Bring the messy version

Secondary:

- Send me what feels unclear
- Ask if this fits your situation
- Tell me what you are trying to clarify

### Website Clarity Review

Primary:

- Send your website for review
- Request a website clarity review
- Check what your website is trying to carry

Secondary:

- Ask what to fix first
- Send the page that feels unclear
- Tell me what the website should explain

### AI Helper Setup

Primary:

- Start with one AI helper
- Plan your AI helper
- Build the first helper role

Secondary:

- Ask what your AI helper should know
- Send the task you want AI to support
- Start with the role, not the prompt

### Brand Clarity Project

Primary:

- Start the brand clarity project
- Request a proposal
- Book the project call

Secondary:

- Ask if this is the right stage
- Send the part that feels scattered
- Start with a clarity session first

### Website Package

Primary:

- Start the website project
- Request a website proposal
- Book the website planning call

Secondary:

- Clarify the site before building
- Ask what your website needs first
- Send your current website

## 5. CTA by page section

### Hero section

Use direct but calm CTAs:

- Book a clarity session
- Start with the red thread
- Send me what feels unclear
- Request a website review
- Start with one helper

### Problem section

Use reflective CTAs:

- Notice what the page is trying to carry.
- Start by naming what feels unclear.
- Bring the messy version.

### Process section

Use next-step CTAs:

- Send the material.
- Book the session.
- Get the clarity base.

### FAQ section

Use reassurance CTAs:

- Ask if this fits.
- Send the question.
- Tell me what is unclear.

### Final section

Use clear invitation CTAs:

- Book a clarity session.
- Start with the red thread.
- Request a proposal.
- Send me what feels unclear.

## 6. CTA by reader state

### Reader feels overwhelmed

Use:

- Bring the messy version.
- Start with one sentence.
- Send me what feels unclear.
- You do not need to tidy it first.

### Reader feels unsure

Use:

- Ask if this fits your situation.
- Send a short note.
- Tell me what you are trying to clarify.
- I can help you choose the next step.

### Reader is ready

Use:

- Book a clarity session.
- Request a proposal.
- Send your website for review.
- Book the project call.

### Reader is comparing options

Use:

- Check if this is the right level of support.
- Start with a clarity session before a bigger project.
- Ask what would be enough for this stage.

## 7. CTA support copy

A CTA often needs one sentence before it.

Examples:

The work does not need to be tidy before we begin.

CTA:

Bring the messy version.

If your website, content, or AI tools need clearer ground, start here.

CTA:

Book a clarity session.

Before you rebuild the page, check what it is trying to carry.

CTA:

Request a website clarity review.

If your AI helper keeps sounding generic, the source material may need work.

CTA:

Start with one helper.

## 8. CTA button text

Use short button text.

Good:

- Book a clarity session
- Send me what feels unclear
- Request a review
- Start with one helper
- Ask if this fits
- Request a proposal
- Book the project call
- Send your website

Avoid:

- Transform my brand
- Yes, I am ready to shine
- Claim my spot
- Unlock clarity now
- Elevate my business
- Scale my visibility
- Take me to the next level
- I’m ready for my dream clients

## 9. CTA endings for emails

Use:

- Reply with what feels unclear.
- Send me the rough version.
- Book the session here.
- Ask if this is the right next step.
- Bring the messy material.
- Start with the red thread.

Avoid:

- Don’t wait.
- Spots are filling fast.
- This is your sign.
- You owe it to yourself.
- It is now or never.

## 10. CTA endings for offer pages

Use:

- Bring the messy version.
- We will find the red thread.
- Start with a clearer base.
- Book a clarity session.
- Ask if this fits your situation.

## 11. CTA endings for checkout pages

Use:

- Complete booking.
- Confirm your session.
- Continue to payment.
- Finish your booking.

Support copy:

After booking, you will receive instructions for sending your material.

## 12. Bad CTA examples

### Bad

Ready to elevate your brand and attract dream clients?

Why it fails:

- vague
- generic
- inflated
- not Luopua

### Better

Ready to make your offer easier to understand?

### Bad

Claim your spot before it is gone.

Why it fails:

- fake urgency unless there is a real limit
- pressure-based
- not grounded

### Better

Book a clarity session.

### Bad

Unlock your full potential with strategic brand copy.

Why it fails:

- inflated
- vague
- says nothing concrete

### Better

Find the sentence your website needs to carry.

## 13. CTA quality check

Before sending CTA options, check:

- Does the CTA match the reader's readiness?
- Is the next step clear?
- Is there any fake urgency?
- Does it sound like Luopua?
- Is it concrete?
- Is it too clever?
- Would Kirsi actually use this?
- Does it help the reader move, not pressure them?

## 14. Final rule

A Luopua CTA is not a push.

It is a clear door.
