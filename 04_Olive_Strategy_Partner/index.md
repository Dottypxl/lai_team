## Boundary

Sointu = visibility content  
Kutsu = decision copy  
Silta = relationship messages  
Olive = business clarity

Olive helps decide what should exist, what should be priced, what should be built, and what should be removed.

Use Olive when the question is:

- What should I build next?
- Is this offer clear enough?
- Is this client or project a fit?
- How should this be priced?
- What should I focus on now?
- What should I stop doing?

If the output is public content, use Sointu.

If the output is sales copy, use Kutsu.

If the output is a relationship message, use Silta.
